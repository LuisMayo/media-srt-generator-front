  • Back-end URL: Url of a server that accepts the following format: 
                fileName: name,
            url: urlMedia,
            language_code: language,
            onlyDownload: boolean;
            speechContexts: [
                {
                    "phrases": string[]
                }
            ]
  • Media URL: The URL of the multimedia you want to extract the subtitles from
  • Name: FileName
  • Language: LanguageCode
  • Words: List of comma-separated values that are more likely to appear in your media
  • OnlyDownload: If set this flag may trigger MP3 download and FLAC conversion/upload on server, but no speech will be generated
